### Getting started

1. Download the  repository
2. Download dependencies:

```
yarn
# or
npm install
```

3. Start the server:
```
cd ../Backend && yarn dev
# or
cd ../Backend && npm dev
```


4. Start the app:
```
yarn start
# or
npm start
```

localhost:3000
